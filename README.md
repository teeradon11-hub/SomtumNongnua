# 🌶️ SomtumNongnua Expense Dashboard

Dashboard สำหรับติดตามค่าใช้จ่ายร้านส้มตำ เชื่อมต่อ Google Sheets แบบ Real-time

## Features
- Dashboard สรุปรายจ่าย
- KPI รายวัน รายเดือน รายปี
- ค้นหาและกรองข้อมูล
- Top Categories / Top Shops
- Local Cache
- Responsive Design
- GitHub Pages Ready

## Deploy
1. แก้ SHEET_ID ใน app.js
2. Push ขึ้น GitHub
3. เปิด Settings > Pages
4. Deploy from branch main
